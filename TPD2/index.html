Deuxieme article

La classe featured ajoute un style special au premier article.

Reponses

  • .main-nav-demo a selectionne tous les liens dans la navigation.
  • .main-nav-demo .active selectionne uniquement le lien actif.
  • .post.featured selectionne l'article qui a la classe featured.
  • .post p:first-of-type selectionne le premier paragraphe de chaque article.
  • .main-nav-demo li:nth-child(even) selectionne les elements li pairs.

Specificite

  1. p < .text < p.text < #main < #main .text p
  2. Si deux regles ont la meme specificite, la derniere declaration gagne.
  3. Il faut eviter !important car il casse la cascade naturelle et complique la maintenance.

Exercice 2

Box Model et centrage

Comparaison

content-box Largeur visible: 350px
border-box Largeur visible: 300px

Reponses

  • Avec content-box, la largeur visible est 350px.
  • Avec border-box, la largeur visible reste 300px.
  • Reset conseille :
*, *::before, *::after {
  box-sizing: border-box;
}
