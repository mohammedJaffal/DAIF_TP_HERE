  • .demo-nav a : tous les liens dans la navigation

Specificite

  1. p < .text < p.text < #main < #main .text p
  2. Si deux regles ont la meme specificite, la derniere declaration gagne.
  3. Il faut eviter !important car il rend le code plus difficile a maintenir.

Exercice 2 - Box Model

Comparaison

content-box Largeur visible : 350px
border-box Largeur visible : 300px

Reponses

  • Avec content-box, la largeur visible est 350px.
  • Avec border-box, la largeur visible reste 300px.
  • Reset conseille :
*, *::before, *::after {
  box-sizing: border-box;
}

Exemple simple de variables CSS :

  --bg-color: #ffffff;
  --text-color: #333333;
  --primary: #007bff;
}

Exercice 8 - Quiz de revision

  1. #header a a la specificite la plus haute.
  2. Avec box-sizing: border-box, la largeur totale reste 200px.
  3. Pour centrer avec Flexbox, il faut justify-content: center et align-items: center.
  4. 1fr signifie une fraction de l'espace disponible.
  5. L'approche mobile first utilise des media queries en min-width.
