# TPD1

Version simple du premier travail pratique HTML/CSS.

## Structure

- `index.html` : page d'accueil avec liens vers les trois sous-parties
- `tp1/` : page de presentation personnelle
- `tp2/` : formulaire HTML5
- `tp3/` : mini site vitrine multi-pages
- `css/style.css` : feuille de style commune
- `js/tp2-validation.js` : validation du formulaire du TP2
- `images/` : illustrations SVG locales
